# pharmacies-de-garde-nouakchott
Application Flutter pour consulter les pharmacies de garde à Nouakchott avec localisation et contact.

# Fonctionnalités  
- Liste des pharmacies de garde en temps réel
- Localisation des pharmacies de garde sur Google Maps
- Recherche par nom de pharmacie
- Affichage des coordonnées (adresse, téléphone)  
- Interface simple et intuitive

# Technologies utilisées 
- Frontend : Flutter(Dart)  
-Backend : Firebase 
- Base de données : Firestore
- API Map : Google Maps API
